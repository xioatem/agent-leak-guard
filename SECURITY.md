# 安全政策 (Security)

`agent-leak-guard` 是一个**个人设备自查与主动防护**工具，不是入侵或攻击工具。

## 报告漏洞

- 工具自身的缺陷（误报 / 漏报 / 兼容性）：请用 `bug_report` / `false_positive` / `false_negative` Issue 模板。
- 涉及敏感披露的：可在 Issue 中注明「私密」，维护者会线下沟通，不在公开区展开。

## 密钥处理

- **报告问题时切勿粘贴真实 API 密钥 / 令牌**。请用明显占位符（如 `xai-XXXXFAKE`）。
- 本仓库 fixture 目录中所有密钥均为伪造占位符。

## 免责

- 本工具**不保证 100% 拦截**所有泄露行为；规则库为社区整理，可能误报或漏报。
- 使用即表示你已知悉并自担风险。
- 严禁将本工具用于未授权扫描、攻击他人系统或任何违法用途。

## 规则来源诚信

所有 `exfil_signatures.json` 条目应来自公开报告或用户侧配置审计。提交新特征时**必须附公开来源**，禁止编造或传播未经证实的指控。
